# AI実践起業塾 - フォント・サイズ・色 一覧表

## フォント設定

### 基本フォント
- **Sans-serif（本文用）**: Noto Sans JP
  - ウェイト: 400, 500, 600, 700
  - 適用: `font-sans`（デフォルト）
  
- **Serif（見出し用）**: Noto Serif JP
  - ウェイト: 400, 500, 600, 700
  - 適用: `font-serif`

## カラーパレット

### 主要カラー
| カラー名 | 用途 | カラーコード（OKLCH） | 説明 |
|---------|------|---------------------|------|
| `background` | 背景 | `oklch(0.97 0.01 90)` | クリーム/オフホワイト |
| `foreground` | テキスト | `oklch(0.15 0 0)` | ダークグレー（ほぼ黒） |
| `primary` | アクセント | `oklch(0.72 0.15 230)` | #38B6FF（スカイブルー） |
| `primary-foreground` | プライマリ上のテキスト | `oklch(1 0 0)` | 白 |
| `muted-foreground` | 補助テキスト | `oklch(0.45 0 0)` | グレー |
| `card` | カード背景 | `oklch(1 0 0)` | 白 |
| `destructive` | 警告・強調 | `oklch(0.577 0.245 27.325)` | レッド系 |

## 使用パターン

### 見出しパターン
1. **ページタイトル**: `font-serif font-bold text-4xl lg:text-5xl text-foreground`
2. **セクションタイトル**: `font-serif font-bold text-3xl lg:text-4xl text-foreground`
3. **カード見出し**: `font-bold text-xl text-foreground`

### 本文パターン
1. **通常本文**: `text-base text-muted-foreground`
2. **大きめ本文**: `text-lg text-muted-foreground`
3. **強調本文**: `text-base text-foreground font-medium`

### アクセントパターン
1. **プライマリ強調**: `text-primary font-bold`
2. **バッジ**: `text-sm font-medium text-primary bg-primary/10`
3. **ボタン**: `text-sm font-medium text-primary-foreground bg-primary`
