# 実績ページの画像アップロード推奨サイズ

## 推奨サイズ

### 横長レイアウト（推奨）
- **サイズ**: 1200px × 600px
- **アスペクト比**: 2:1
- **形式**: JPG, PNG, WebP
- **最大ファイルサイズ**: 5MB

### その他の対応サイズ
- 1600px × 800px（2:1）
- 1000px × 500px（2:1）
- 800px × 400px（2:1）

## 重要なポイント

1. **アスペクト比を維持**
   - 2:1（横:縦）の比率を維持することで、画像が適切に表示されます
   - 他の比率の画像も表示できますが、上下が切れる可能性があります

2. **ファイルサイズ**
   - 5MB以下に抑えてください
   - 画像を圧縮することで、ページの読み込み速度が向上します

3. **解像度**
   - Retinaディスプレイ対応のため、実際の表示サイズの2倍の解像度を推奨
   - 表示サイズが600px幅の場合、1200pxの画像を使用

4. **推奨形式**
   - **WebP**: 最も軽量で高品質（推奨）
   - **JPG**: 写真に適している
   - **PNG**: 透明背景が必要な場合

## 画像の準備方法

### オンラインツール
1. **TinyPNG** (https://tinypng.com/)
   - PNG/JPGを圧縮
   
2. **Squoosh** (https://squoosh.app/)
   - WebPに変換・圧縮

3. **Canva** (https://www.canva.com/)
   - カスタムサイズ: 1200 × 600px で作成

### 画像編集ソフト
1. **Photoshop**
   - Web用に保存: 品質80-90%
   
2. **GIMP**（無料）
   - エクスポート時に品質を調整

## 表示例

実績ページでは、以下のように表示されます：

```
┌─────────────────────────────────────┐
│                                     │
│         画像（1200×600px）          │
│         高さ300pxで表示             │
│                                     │
└─────────────────────────────────────┘
```

- **表示高さ**: 300px（固定）
- **表示幅**: コンテナの幅に合わせて自動調整
- **オブジェクトフィット**: `cover`（画像が枠内を埋めるように表示）

## トラブルシューティング

### 画像が表示されない場合

1. **ファイルサイズを確認**
   - 5MB以下か確認

2. **ファイル形式を確認**
   - JPG, PNG, WebPのいずれかか確認

3. **再アップロード**
   - 画像を削除して再度アップロード

4. **ブラウザキャッシュをクリア**
   - Ctrl+Shift+R（Windows）
   - Cmd+Shift+R（Mac）
